## 安装部署

### Q: 首次访问系统显示什么？

首次访问会自动跳转到安装向导页面，需要配置：
1. MySQL 数据库连接信息（主机、端口、数据库名、用户名、密码）
2. 管理员账号信息（用户名、密码、姓名）

### Q: 支持哪些数据库？

目前支持 MySQL 8.0+ 数据库，包括：
- 本地 MySQL 服务器
- 云数据库（如阿里云 RDS、腾讯云 MySQL 等）
- Docker 容器中的 MySQL

### Q: 数据库连接失败怎么办？

检查以下项目：
1. 数据库服务是否已启动
2. 连接信息是否正确（主机、端口、用户名、密码）
3. 数据库用户是否有足够权限（需要 CREATE、ALTER、INSERT、UPDATE、DELETE 权限）
4. 防火墙是否允许连接
5. 如果是云数据库，检查白名单设置

### Q: 安装后如何修改数据库配置？

数据库配置保存在项目根目录的 `db.config.json` 文件中，可以直接编辑该文件修改配置。

### Q: 如何重新安装？

1. 删除 `db.config.json` 文件
2. 清空数据库中的所有表
3. 重新访问系统，会自动进入安装向导

## 登录与认证

### Q: 忘记管理员密码怎么办？

**方法一：API 重置（推荐）**
```bash
curl -X POST http://localhost:5000/api/admin/reset-password
```
新密码会保存到 `su.passwd` 文件中。

**方法二：管理员重置**
使用其他管理员账号登录，在「用户管理」页面重置密码。

### Q: 登录时提示"账号已被锁定"怎么办？

账号在连续多次登录失败后会自动锁定。解决方法：
1. 等待锁定时间结束（默认 30 分钟）
2. 管理员在用户管理页面解锁账号
3. 重置用户密码

### Q: 为什么登录后跳转回登录页？

可能的原因：
1. Cookie/SessionStorage 被浏览器阻止
2. 登录状态已过期
3. 跨域访问导致认证信息丢失

解决方法：
1. 检查浏览器是否禁用了 Cookie
2. 清除浏览器缓存后重新登录
3. 如果是跨域访问，确保使用 HTTPS 协议

### Q: 登录时需要验证码是怎么回事？

系统安全策略：首次登录无需验证码，密码错误 3 次后才会显示验证码。这是为了防止暴力破解攻击。

### Q: 首次登录为什么要修改密码？

为了安全考虑，管理员创建的用户或重置密码后的用户，首次登录都必须修改密码。这确保只有用户本人知道自己的密码。

## 数据库相关

### Q: 如何备份数据？

```bash
# 使用 mysqldump 备份
mysqldump -u root -p database_name > backup.sql

# 备份指定表
mysqldump -u root -p database_name users products inventory > backup_tables.sql
```

### Q: 如何恢复数据？

```bash
# 恢复数据库
mysql -u root -p database_name < backup.sql
```

### Q: 数据库表结构在哪里？

表结构由系统自动创建。安装时会在数据库中创建以下主要表：
- `users` - 用户表
- `products` - 商品表
- `categories` - 分类表
- `warehouses` - 仓库表
- `inventory` - 库存表
- `sales_orders` - 销售订单表
- `sales_order_items` - 销售订单明细表
- `customers` - 客户表
- `purchase_orders` - 采购订单表

### Q: 如何查看数据库当前状态？

访问 API 接口：
```bash
curl http://localhost:5000/api/init/status
```

## 功能使用

### Q: 如何创建销售单？

1. 登录系统后进入「销售管理」页面
2. 点击「新建销售单」按钮
3. 选择仓库
4. 填写客户信息（姓名和手机号为必填项）
5. 添加商品，系统会自动计算金额
6. 确认提交

### Q: 销售时客户信息怎么填写？

客户信息字段说明：
| 字段 | 是否必填 | 说明 |
|------|---------|------|
| 姓名 | 是 | 客户姓名 |
| 手机号 | 是 | 用于联系和识别客户 |
| 微信号 | 否 | 微信联系方式 |
| 地址 | 否 | 收货地址 |

系统会自动匹配已有客户，如果是新客户会自动创建。

### Q: 如何管理多仓库？

1. 进入「仓库管理」页面
2. 可以添加、编辑、删除仓库
3. 创建销售单或采购单时可以选择不同仓库
4. 库存会按仓库分别统计

### Q: 库存预警是什么？

当商品库存低于设定的最小库存值时，系统会在首页显示库存预警提醒，方便及时补货。

### Q: 如何切换主题色？

点击页面右上角的调色板图标，可以选择 8 种预设主题色：
- 海洋蓝、优雅紫、森林绿、活力橙
- 热情红、浪漫粉、青碧色、靛蓝色

### Q: 如何切换语言？

点击页面右上角的语言选择器，支持：
- 简体中文
- 繁体中文
- English
- 日本語

## 错误排查

### Q: 页面显示空白或加载不出来？

检查步骤：
1. 打开浏览器开发者工具（F12）查看控制台错误
2. 检查网络请求是否正常
3. 清除浏览器缓存
4. 检查服务是否正常运行

```bash
# 检查服务状态
curl -I http://localhost:5000
```

### Q: API 请求返回 500 错误？

检查步骤：
1. 查看服务器日志：`/app/work/logs/bypass/app.log`
2. 检查数据库连接是否正常
3. 检查请求参数是否正确

### Q: 端口被占用怎么办？

```bash
# 查看 5000 端口占用
ss -lptn 'sport = :5000'

# 终止占用进程
kill -9 <PID>
```

### Q: 构建失败怎么办？

常见原因：
1. 依赖安装不完整 → 运行 `pnpm install`
2. Node.js 版本不兼容 → 使用 Node.js 18+
3. 内存不足 → 增加可用内存

### Q: 热更新不生效？

1. 检查是否使用 `coze dev` 或 `pnpm dev` 启动
2. 尝试手动刷新页面
3. 重启开发服务

### Q: 主题切换没有效果？

1. 清除浏览器缓存
2. 检查浏览器是否支持 CSS 变量
3. 确认使用的是现代浏览器（Chrome、Firefox、Edge、Safari 最新版本）
