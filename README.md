# 零镜进销存

一个专业的企业级进销存管理系统——零镜进销存，支持多用户、多仓库管理，具备完整的商品、采购、销售、库存、客户管理功能。

## 功能特性

### 核心功能

- **商品管理** - 商品信息维护、分类管理、SKU编码
- **采购管理** - 采购订单创建、入库管理
- **销售管理** - 销售订单处理、客户信息录入
- **客户管理** - 客户信息维护、购买记录追踪
- **库存管理** - 多仓库库存监控、库存预警
- **报表统计** - 销售报表、采购报表、数据分析

### 系统特性

- **多用户权限** - 支持管理员、经理、普通用户三种角色
- **多仓库管理** - 支持多仓库协同运营
- **主题定制** - 8种预设主题色，一键切换
- **多语言支持** - 简体中文、繁体中文、English、日本語
- **安全防护** - 密码加密、登录验证码、权限控制

## 快速开始

### 环境要求

- Node.js 18+
- MySQL 8.0+
- pnpm (推荐)

### 安装部署

1. **克隆项目**
```bash
git clone https://github.com/geronzhang-lgtm/ZeroMirror.git
cd inventory-system
```

2. **安装依赖**
```bash
pnpm install
```

3. **启动开发服务**
```bash
coze dev
```

4. **访问系统**

打开浏览器访问 `http://localhost:5000`，按照安装向导完成系统初始化。

### 生产部署

```bash
# 构建
coze build

# 启动
coze start
```

## 功能模块

### 用户管理

| 角色 | 权限 |
|------|------|
| 管理员 (admin) | 全部权限，包括用户管理、系统配置 |
| 经理 (manager) | 商品、采购、销售、库存管理 |
| 普通用户 (user) | 查看权限，创建销售单 |

### 销售管理

- 客户信息录入（姓名、手机号必填，微信号、地址选填）
- 商品搜索和选择
- 实时计算订单金额
- 自动扣减库存

### 客户管理

- 客户信息维护
- 购买记录追踪
- 消费统计
- 多条件搜索（姓名、手机号、微信号、地址）

### 库存管理

- 多仓库支持
- 库存实时更新
- 库存预警提醒
- 批量导入导出
